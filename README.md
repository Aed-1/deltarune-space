# deltarune-space
I noticed the in game settings menu in deltarune doesnt allow you to set keys like spacebar or any special character as the "enter" or "continue" button, which really bugged me since i use spacebar for every game like this, so i made a progam to do it for me! The code is fully open source so if your worried about getting hacked or anything, you can look through the code, its pretty simple.

If you encounter any issues or bugs, report them to me at \_aed_ on discord. please message me when you add me, otherwise i may decline the request thinking you are a bot

#GUIDE
here is the guide for the EXE file (the simplest option)

1. go to releases on this repo
2. go to the most recent release
3. download the .EXE file
4. run the file
5. select the option you want
6. enjoy!!
   
#GUIDE FOR PYTHON FILE
here is the guide for running the script through the raw python file

1. download the .py file off this repo
2. do ```pip install psutil``` and ```pip install pywin32``` in command prompt, this will install the dependancies
3. run the .py file
4. select the option you want
5. enjoy!!
